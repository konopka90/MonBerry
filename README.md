# raspberry-enviro

## How to run

```bash
docker compose up -d
```

## How to kill

```bash
docker compose down
```

## How to kill and remove data

```bash
docker compose down --volumes
```

## Development

Create and activate env

```bash
python3 -m venv venv
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```
